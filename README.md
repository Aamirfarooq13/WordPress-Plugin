# WordPress-Plugin
Text Animation Plugin for WordPress 6.9+. Use the `[text_animation]` shortcode to animate text with a letter-by-letter effect.

## Shortcode usage
```
[text_animation text="Animate this headline" effect="fade" speed="60"]
```

### Attributes
- `text`: The text to animate. You can also wrap content inside the shortcode.
- `effect`: `fade` (default), `typewriter`, or `bounce`.
- `speed`: Delay in milliseconds between letters (default `60`).
